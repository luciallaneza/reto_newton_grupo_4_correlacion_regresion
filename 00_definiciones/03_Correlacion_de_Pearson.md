# Correlación de Pearson

La matriz de correlación de Pearson es una tabla que resume los coeficientes de correlación lineal entre múltiples variables cuantitativas, indicando la fuerza y dirección de sus relaciones. 

Los valores oscilan entre -1 (relación negativa perfecta) y +1 (positiva perfecta), donde 0 indica ausencia de relación lineal. 

Es simétrica, con unos "1" en la diagonal. 

## Conceptos Clave
**•	Finalidad:** Medir el grado de asociación lineal entre pares de variables numéricas (intervalo o razón).

**•	Rango de Valores (r):**

    o	+1: Correlación positiva perfecta.
    o	0: No hay relación lineal.
    o	-1: Correlación negativa perfecta.
    o	|r| < 0.3: Correlación débil
    o	0.3 ≤ |r| < 0.7: Correlación moderada
    o	|r| ≥ 0.7: Correlación fuerte
**•	Interpretación:** Cerca de 1 o -1 indica una relación fuerte, mientras que cerca de 0 indica una relación débil.

**•	Estructura:** Tabla simétrica donde la intersección de fila y columna muestra la correlación (rxy) entre dos variables. 


## Suposiciones y Limitaciones
Para que la correlación de Pearson sea válida, los datos deben cumplir ciertos requisitos:

**•	Linealidad:** La relación entre variables debe ser lineal.   
**•	Normalidad:** Los datos deben seguir una distribución normal bivariada.   
**•	Variables Numéricas:** Solo funciona con datos cuantitativos continuos.   
**•	Sensibilidad:** Es muy sensible a valores atípicos (outliers), que pueden distorsionar los resultados
