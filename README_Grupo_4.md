## Trabajo de equipo

En nuestro equipo nos hemos encargado de investigar:

- Consumo de la API pública [REST Countries](https://restcountries.com).
- Correlación-regresión (Pearson y Spearman).   
- Regresión lineal simple.
    - Pendiente (β₁)
    - Intercepto (β₀)
    - Bondad de ajuste (R²)
- Residuos

## Punto de inicio ╰┈➤
REST Countries API   
URL: https://restcountries.com/   
Versión: v3.1   
Tipo: API REST pública y gratuita   
Endpoint utilizado:   
https://restcountries.com/v3.1/all?fields=name,area,population,region,gini   

<u>Datos obtenidos:</u>  
Nombre del país   
Región geográfica   
Población   
Área (km²)

<u>Uso en el proyecto:</u>   
 Fuente de datos principal para el análisis de correlación y regresión lineal entre área y población de una muestra de 100 países.   
<u>Autenticación:</u> No requiere API key ✅   
<u>Formato de respuesta:</u> JSON
